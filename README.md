# Debt Help Landing Page

A React.js landing page for Debt Help Canada.

## Features

- Responsive Landing Page
- Multi-step Assessment Form
- UTM Tracking
- Facebook _fbc Cookie Tracking
- React Router Navigation
- Hero Section
- How It Works
- Debt Types
- Testimonials
- Footer

## Installation

npm install

npm run dev

## Technologies

- React.js
- Vite
- CSS3
- React Router
